== Introduction

This is a self-paced lab, aimed at developers using, or wanting to use, Red Hat JBoss Enterprise Application Platform 7.

The lab consists of several exercises walking the participant through basic and new features of Red Hat JBoss EAP 7.

Here is a short list for the available exercises:

* Exercise 1 - Environment setup
* Exercise 2 - Java EE 7 Front-end technologies
* https://github.com/tqvarnst/jboss-eap7-summit-lab/blob/master/exercise3-batch.adoc[Exercise 3 - Batch Applications]
* https://github.com/tqvarnst/jboss-eap7-summit-lab/blob/master/exercise4-messaging.adoc[Exercise 4 - Messaging with ActiveMQ Artemis]
* https://github.com/tqvarnst/jboss-eap7-summit-lab/blob/master/exercise5-concurrency.adoc[Exercise 5 - Concurrency Utilities]
* Exercise 6 - Load-balancing
* Exercise 7 - Clustering
